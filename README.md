# Dock Lift

Développée en Go 🐹, ce projet permet de créer des backups S3 ☁️ de vos fichiers stockés dans des conteneurs Docker 🐳. Pensé pour être déployé rapidement partout

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/jobiyax/dock-lift.git

# Accéder au dossier
cd dock-lift

# Nettoyer les dépendances
go mod tidy

# Lancer le projet
go run main.go
```
